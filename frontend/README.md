# 前端模块  

插件应存放的位置：statics/plugins/  

## 目录结构  

```text  
frontend                                     #前端模块根目录
    |- statics                               #静态资源文件夹
        |- css                               #样式文件夹
            *.css                            #CSS样式文件
        |- js                                #JavaScript文件夹
            common.js                        #前端模块JavaScript代码文件
        |- img                               #图片素材文件夹
            *.*                              #图片素材
                ...                          #某插件包含的文件
    *.html                                   #HTML文件
```
